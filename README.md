# ResolutionGameDevW4
This is an rpg game made during Resolution Game Dev Week 4 with GameMaker!

I learned how to combat and have a health bar. This is my second project in GameMaker. 

I did use premade assets, however I modified and added my own!

Watch Lapse [here](https://lapse.hackclub.com/timelapse/XvmQw6wKCISw).

### How to Play:
Use WASD to move and press SPACE to interact/attack. Collect all keys before running out of battery. Once you collect all the keys, you can enter your house and recharge. 

Play [here!](https://booleanbunny14.itch.io/resolutiongamedevw4)
